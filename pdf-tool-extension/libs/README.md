# Required Libraries

Download and place the following libraries in this folder:

## PDF-lib (v1.17.1+)
Download from: https://unpkg.com/pdf-lib@1.17.1/dist/pdf-lib.min.js
Save as: `pdf-lib.min.js`

## PDF.js (v4.0+)
Download from: https://unpkg.com/pdfjs-dist@4.0.379/build/pdf.min.mjs
Save as: `pdf.min.mjs`

Download from: https://unpkg.com/pdfjs-dist@4.0.379/build/pdf.worker.min.mjs
Save as: `pdf.worker.min.mjs`

## Quick Download Commands

```bash
curl -L -o pdf-lib.min.js "https://unpkg.com/pdf-lib@1.17.1/dist/pdf-lib.min.js"
curl -L -o pdf.min.mjs "https://unpkg.com/pdfjs-dist@4.0.379/build/pdf.min.mjs"
curl -L -o pdf.worker.min.mjs "https://unpkg.com/pdfjs-dist@4.0.379/build/pdf.worker.min.mjs"
```
