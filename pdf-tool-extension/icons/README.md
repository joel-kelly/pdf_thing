# Extension Icons

The current icons are simple placeholders. For a better user experience, replace them with custom designed icons.

## Required Sizes
- `icon-48.png` - 48x48 pixels (toolbar icon)
- `icon-96.png` - 96x96 pixels (high-DPI toolbar icon)

## Design Suggestions
- Use a PDF document icon with editing indicators
- Color scheme: Red (#e74c3c) works well for PDF tools
- Keep the design simple and recognizable at small sizes
- Use transparent backgrounds where appropriate

## SVG Source Files
The SVG files in this folder can be used as a starting point for creating better icons using a vector graphics editor.
